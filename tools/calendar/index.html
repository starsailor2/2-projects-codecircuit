Plan your week with draggable task blocks

Keyboard Shortcuts

  • Ctrl + E - Create event (when time slot is selected)
  • Ctrl + ← - Previous week
  • Ctrl + → - Next week
  • Ctrl + T - Go to current week
  • Ctrl + C - Focus on current time
  • Esc - Cancel selection
  • Click on "+" - Create new event
  • Right-click event - Show context menu (Edit/Delete)
  • Double-click event - Edit event
  • Drag event - Move to new time/day
  • Drag bottom edge - Resize event duration
  • Long-press (mobile) - Open context menu
  • Swipe left/right - Change weeks
